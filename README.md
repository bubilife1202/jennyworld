# Jennyworld

Roblox-inspired mobile web escape room prototype built with `PlayCanvas Engine + TypeScript + Vite`.

## Current MVP

- One playable stage: `무지개 교실`
- Third-person 3D room with blocky Roblox-style character
- Four child-friendly puzzles
- Mobile joystick + action button overlay
- Progress saved in `localStorage`

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy To GitHub Pages

1. Push this project to a GitHub repository.
2. In GitHub, open `Settings > Pages`.
3. Set `Build and deployment` to `GitHub Actions`.
4. Push to `main` and wait for the `Deploy GitHub Pages` workflow to finish.
